Welcome to the Tetra Tech Watershed Boundary Dataset Tool

Python Django Rest Framework API implementation of the Hydrologic Unit Code (HUC) and Watershed Boundary Dataset. Provides a complete API to navigate the HUC/WBD dataset(s) in JSON and XML formats. Two sample clients are also included. 'Chart' provides a D3js implementation of a WBD browser. 'Map' provides a map-based implementation of a WBD navigation too.

The code for the Watershed Boundary Dataset Tool is hosted in GitHub.

https://github.com/JamesBisese/wbd.git
